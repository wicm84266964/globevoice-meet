# 在 Zoom 视频会议中使用实时翻译

## 概述

要让 Zoom 会议中的其他人听到翻译后的目标语言语音，你需要使用**虚拟音频设备**。这样翻译后的音频会输出到虚拟设备，然后你可以在 Zoom 中选择这个虚拟设备作为麦克风输入。

GlobeVoice Meet 不会随发布包内置或安装虚拟音频插件。请从官方渠道自行下载并安装 VB-Cable、Voicemeeter、BlackHole 等第三方虚拟音频设备。

## Windows 系统设置步骤

### 1. 安装虚拟音频设备

推荐使用以下虚拟音频设备软件：

> 出于驱动安装权限、系统兼容性和第三方授权原因，本项目不打包分发这些插件；请优先使用下方官方链接。

#### 选项 A: VB-Cable（免费）
- 下载地址: https://vb-audio.com/Cable/
- 下载并安装 VB-Audio Cable
- 安装后会在系统中创建一个虚拟音频设备 "CABLE Input"

#### 选项 B: Voicemeeter（免费，功能更强大）
- 下载地址: https://vb-audio.com/Voicemeeter/
- 下载并安装 Voicemeeter（标准版即可）
- 安装后会在系统中创建虚拟音频设备

### 2. 配置翻译软件

1. 启动 GlobeVoice Meet，进入**音频设备设置**
2. **输入设备**：选择你的真实麦克风
3. **输出设备**：选择虚拟音频设备（例如 "CABLE Input"）
4. 点击**启动服务**

> 你也可以在前端界面中选择**源语言**和**目标语言**，默认为中文 → 英文。

### 3. 配置 Zoom

1. 打开 Zoom 设置
2. 进入"音频"设置
3. 在"麦克风"下拉菜单中选择虚拟音频设备（例如 "CABLE Output"）
4. 在"扬声器"中保持你的真实扬声器/耳机

## 工作原理

```
你的麦克风 → 翻译软件 → 虚拟音频设备 → Zoom（其他人听到目标语言语音）
                                    ↓
                              你的耳机（你听到目标语言语音）
```

## 注意事项

1. **音频延迟**: 虚拟音频设备可能会增加一些延迟
2. **回声问题**: 可能需要在 Zoom 中开启回声消除
3. **设备名称**: 不同虚拟音频软件的设备名称不同，请根据实际情况选择
4. **测试**: 在正式会议前先测试，确保对方能清晰听到翻译后的语音

## 故障排除

### 问题 1: Zoom 中听不到翻译声音
- 检查虚拟音频设备是否正确安装
- 确认 Zoom 中选择的麦克风是虚拟设备
- 检查翻译软件的状态信息，确认音频正在输出到正确的设备

### 问题 2: 音质不好或有杂音
- 尝试调整虚拟音频设备的采样率设置
- 检查系统音频驱动是否最新
- 尝试不同的虚拟音频软件

### 问题 3: 延迟太高
- 检查网络连接
- 考虑使用有线网络而非 WiFi
- 关闭其他占用网络的应用程序
